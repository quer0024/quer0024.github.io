HTML Tags I've learned

em
The "emphasis" tag highlights a word by turning into italic text like this. This can also be done using "i".
hr
The "hr" tag adds a horizontal line on a page.
ol
The "ol" tag creates an ordered list using numbers starting at 1.
img src="..."
This tag allows you to add an image from a source. You should also add alternate text with the "alt" tag and a title of the image using the "title" tag. See an exmaple of an image on the home page.
table
The "table" tag allows you to create a table with varying information.
